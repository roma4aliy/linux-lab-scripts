# Linux Lab Scripts

Репозиторій містить виконані лаб. роботи

## Вміст

- 'count_files' - скрипт для підрахунку файлів у /etc
- 'packaging` - файли для створення RPM та DEB пакетів
- `github/workflows/` - GitHub Action для автоматичної збірки пакетів
- `Jenkinsfile` - Jenkins Pipeline для CI/CD

## Використання

'''bash
./count_files
'''
